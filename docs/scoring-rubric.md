# Scoring Rubric

Overall score:

```text
overall_score = static_score * 0.35 + benchmark_score * 0.65
```

No token budget scoring is included.

Static score covers:

- structure integrity
- trigger clarity
- task boundary
- workflow clarity
- verifiability
- support materials
- link health

Benchmark score covers:

- scenario completion
- verifier pass rate
- success checklist pass rate
- output quality
- tool call quality
- boundary handling
- stability
